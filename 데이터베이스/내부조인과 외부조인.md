### 내부 조인(Inner Join)
```sql
SELECT * FROM TableA A
INNER JOIN TableB B ON
A.key = B.key
```

### 외부 조인(Outer Join)
- Left, Full, Rgith (Left Join이 Left Outer Join 줄인 말이다, Right도 같은 패턴)
- 단, Full은 Outer를 줄이지 않고 그대로 쓰는 듯하다.
- 나머진 SQL Joins Visualizer에서 상세히 확인 가능하다.
