`알고리즘 수업 - 버블 정렬 1` <br>
https://www.acmicpc.net/problem/23968 <br>
```python
import sys
input=sys.stdin.readline
N,K=map(int,input().split())
a=list(map(int,input().split()))
cnt=0
#multiple cycle
for cycle in range(len(a)-1):
    #1cycle
    for i in range(len(a)-1-cycle): #가장 큰 값 1개는 맨 뒤로. (-cycle)
        if(a[i]>a[i+1]):
            #swap
            a[i],a[i+1]=a[i+1],a[i]
            cnt+=1
            if(cnt==K):
                print(a[i], a[i+1])
                
#교환 횟수<K  return -1
if(cnt<K):
    print(-1)
```
