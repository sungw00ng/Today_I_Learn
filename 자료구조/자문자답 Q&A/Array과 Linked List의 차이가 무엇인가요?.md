# 배열과 연결리스트의 차이가 무엇인가요?
```txt
배열은 메모리 상에 연속된 공간에 데이터를 저장해서
인덱스로 빠르게 접근할 수 있는 자료구조입니다.
그래서 특정 위치의 원소를 접근하는 데는
O(1)의 시간복잡도를 가집니다.

하지만,
배열은 크기가 고정되어 있거나,
동적 배열인 경우에도 용량이 꽉 차면
새 공간을 할당하고데이터를 복사해야 해서
삽입이나 삭제가 비효율적일 수 있습니다.

반면에,
연결 리스트는 각 노드가
데이터와 다음 노드를 가리키는 포인터를 가지고 있어
메모리 상에 비연속적으로 존재할 수 있습니다.

따라서,
삽입이나 삭제 시 포인터만 조작하면 되어
중간에 데이터를 추가하거나 삭제하는 데 용이하지만,
인덱스로 임의 접근이 불가능해 원하는 위치까지
순차 탐색해야 하므로
접근 속도가 O(n)입니다.

정리하자면,
빠른 임의 접근이 필요하고
메모리 연속성이 중요하면 배열을 사용하고,
삽입과 삭제가 자주 발생하는 경우에는
연결 리스트를 사용하는 것이 효율적입니다."
```
